# acc-bot

## Setup

### What You'll Need
- Node.js (pref. v12)
- Yarn

### Installing dependencies
1. `yarn install`

### Environment Variables
1. Create `.env` file in the project root, to be filled in with secret keys later, with the following starter code
```
BOT_TOKEN = 

FIREBASE_API_KEY             = 
FIREBASE_PROJECT_ID          = 
FIREBASE_MESSAGING_SENDER_ID = 
FIREBASE_APP_ID              = 
```
